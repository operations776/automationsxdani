# Daniyal Aziz · Portfolio

Personal site of Muhammad Daniyal Aziz: GTM Engineer and AI Automation Specialist at RecruiterGTM.

Not a brochure. The hero is an animated map of a Claude ops manager working a recruitment client's stack in real time, and every case study is a working miniature: watch a sourcing run happen step by step, run the cross-channel reply protection flow, click through the Pulse Tracker product, book a fake call in the in-house booking engine, and assemble outbound angles by ICP. No code on screen, just the systems doing their jobs.

Live: https://daniyal1234-alt.github.io/daniyal-aziz-protfolio/

## Stack

Vite · React 18 · TypeScript · Tailwind CSS · shadcn/ui. Deployed on GitHub Pages.

## Develop

```
npm install
npm run dev       # local dev server on :4242
npm run build     # production build to dist/
npm run deploy    # build + publish to GitHub Pages
```

## Structure

- `src/components/hero-visual.tsx` : the animated orchestration map in the hero
- `src/components/demos/` : interactive case-study miniatures
- `src/components/*-section.tsx` : page sections
- `public/logos/` : company and tool logos
- `REDESIGN-PLAN.md` : the plan behind the 2026 redesign
